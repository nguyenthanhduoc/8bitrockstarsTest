
### To run local
```
$ git clone https://github.com/nguyenthanhduoc/8bitrockstarsTest.git
$ cd 8bitrockstarsTest
$ npm install
$ npm run dev
$ Go to: http://localhost:8080/
```

### Prerequisite
- NodeJS (v8.3.1)
- NPM (v5.0.3)
